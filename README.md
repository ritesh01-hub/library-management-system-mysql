
# Library Management System (MySQL)

A Library Management System built using MySQL to manage books, members, and issue/return operations with proper relational database design.

## Features
- Book management
- Member management
- Issue and return books using SQL transactions
- Overdue book tracking
- Relational database design using foreign keys

## Technologies Used
- MySQL

## Concepts Covered
- SQL JOINs
- Transactions (ACID properties)
- Constraints & Foreign Keys
- Normalization
- Aggregate functions

## Future Enhancements
- Backend integration using Node.js or Flask
- REST API implementation
- User interface for library operations
